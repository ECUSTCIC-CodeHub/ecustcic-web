# 快速入门指南

## 快速修改配置
### 电竞热门游戏
```
/src/data/esportsGames.js
```

### 交流群
```
/src/data/qqGroups.js
```

### 文档菜单
```
/src/data/menu.js
```

## 快速开始
1. 安装依赖
```
npm install
```

2. 启动开发服务器
```
npm run serve
```

## 其他命令（可选）
- 构建生产版本: ```npm run build```
- 运行测试: ```npm run test```
